**React application for store email address**

author: Ivan Zavadsky

**Local setup:**
--
1. `git clone`
2. `npm install`
3. create `src/config/dev.js` and fill it in with your credits.
4. `npm start`



**Production build:**
--

1.  create `src/config/prod.js` and fill it in with your credits.
2. `npm run build`
